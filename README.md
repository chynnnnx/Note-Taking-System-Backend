# Note-Taking System Backend

## Overview
This is the backend for a full-stack **Note-Taking System** built with .NET, following **Clean Architecture**, **CQRS**, and the **Mediator pattern**.  
The backend handles authentication, API endpoints, and data management for the Angular frontend.

---

## Features
- **User Authentication:** JWT-based login and registration.  
- **Notes Management:** Create, read, update, and delete notes.  
- **CQRS & Mediator Pattern:** Ensures scalable and maintainable code.  
- **Clean Architecture:** Separation of concerns for better testing and maintenance.  
- **API Endpoints:** Fully functional REST APIs for frontend integration.  

---

## Tech Stack
- **Backend:** .NET 9, C#  
- **Patterns:** Clean Architecture, CQRS, Mediator  
- **Database:** SQL Server  (MSSQL)
- **Authentication:** JWT  
- **Frontend:** Angular (frontend not included in this repo)
